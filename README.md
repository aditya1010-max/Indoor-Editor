# Indoor Graph Editor

A lightweight web application for creating topological indoor floorplans using D3.js.

- **Browse Mode**: Pan and zoom across the canvas to navigate your layout.
- **Draw Mode**: Click to place nodes and automatically connect them into "Ways."
- **Snapping**: Smart node-to-node snapping for perfect geometric connectivity.
- **Built with**: Vanilla JS, D3.js, and Vite for a fast local development experience.
